## Coverage definition

- **direct**: SDK contains `functionName: '<fn>'` or `encodeFunctionData(..., '<fn>')` (a deliberate wrapper).
- **indirect/verify**: only a bare `.<fn>(` method-call reference — counted as covered but ambiguous (generic JS names collide); should be confirmed manually.
- **gap**: no SDK reference to the name at all.
- **KMS**: an endpoint is covered when a kms-* service references its exact path string. `GET /` and `GET /test` (HTML pages) are exempt.
